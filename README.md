# Pizza-Sales-SQL
This project analyzes pizza sales data using SQL to gain insights into sales performance, customer ordering patterns, and revenue trends.
The dataset contains information about orders, pizzas, pizza types, and order details.
Dataset Description
The project uses the following CSV files:
orders.csv – Contains order date and time information
order_details.csv – Contains details of pizzas ordered (quantity, pizza ID)
pizzas.csv – Contains pizza size and price information
pizza_types.csv – Contains pizza name, category, and ingredients
